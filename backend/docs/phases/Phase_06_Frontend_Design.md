# Phase 6: Frontend - Layout & Design System

## 🎯 Goal
Build the visual foundation of the application with a premium, glassmorphic look.

## 🛠️ Tasks

### 1. Project Scaffolding
- Set up a basic HTML5 structure.
- Integrate **Tailwind CSS** (via CDN for simplicity or PostCSS for performance).

### 2. Design Tokens (CSS)
- Define the "Glassmorphism" utility classes:
  - `bg-white/5`
  - `backdrop-blur-xl`
  - `border-white/10`
- Implement the deep navy background: `bg-slate-950`.

### 3. Layout Shelves
- **Header**: Logo and user profile toggle.
- **Hero Section**: Large search input centered on the screen.
- **Results Area**: Responsive grid (1 col mobile, 3 col desktop).

### 4. Navigation Flow
- Implement "Login/Signup" modals or separate views.
- Implement the "Protected View" (results container) that only shows after login.

## ✅ Success Criteria
- [ ] UI looks premium and follows the Dark Mode Glassmorphic theme.
- [ ] Navbar and Search Bar are fully responsive.
- [ ] Modals for Login/Signup functional and centered.
